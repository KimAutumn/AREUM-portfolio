Portfolio Website
김아름 포트폴리오 웹사이트
HTML, SCSS, JQeury
OVERVIEW
웹 디자이너이자 캐릭터 디자이너인 나 자신 '김아름'을 소개하고
디자이너로서 그동안 작업해온 결과물들을 보여줄 수 있는 포트폴리오 웹사이트이다.
나만의 개성을 표현하기 위해 캐릭터 디자이너이자 일러스트레이터라는 특징을 살려 캐릭터와 일러스트를 그려 홈페이지를 구성했다.
INTRODUCE
나에 대한 간단한 소개와 학력, 경력, 그리고 툴에 대한 능력치를 정리해 슬라이드 형식으로 볼 수 있도록 하였다.
CATEGORIES
다양한 디자인 능력을 상세하게 보여줄 수 있도록 크게 Web / Character / Graphic 으로
페이지를 나누어 각각 세 페이지에 해당하는 작품들을 분류해 넣어 깔끔하게 정리하였다.
DETAILS
마지막으로 이 포트폴리오 사이트를 본 사람이 나를 마음에 들어할 때 언제든지 연락할 수 있도록
연락처 정보를 기입하고 나의 이력서, 포트폴리오(pdf)를 다운받아 열람할 수 있는 'Contact'페이지를 만들었다.
